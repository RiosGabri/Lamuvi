Termos de Serviço (MVP)

Este documento estabelece as regras de uso para a versão de testes do Lamuvi.

1. Objetivo do Teste

O Lamuvi está em fase de MVP (Minimum Viable Product). O objetivo atual é validar as funcionalidades de cadastro, login e listagem de filmes.

2. Uso de Dados

Ao se cadastrar, você concorda que seus dados (nome e senha) sejam armazenados temporariamente para fins de teste de autenticação no sistema.

3. Responsabilidades

Como este é um ambiente de desenvolvimento, não recomendamos o uso de senhas reais ou importantes. O sistema pode passar por reformulações a qualquer momento.
